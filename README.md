# Phone-Catalogue

Tools

---

Key tools used in this project are:

- ES6
- Bootstrap
- Reactstrap
- Express
- React
- Redux

# Installation

---

- On the app folder:

Execute the following command on your terminal to install all the needed packages:

    npm install

To build the React app:

    npm build

To start the app:

    npm run start

The API will run on port 3001 and the app on port 3000
